# Flight Time Tracker

## Автор

Большаков Дмитрий

Тестовое задание **Java SE**

Необходимо реализовать приложение, которое загружает и обрабатывает информацию о рабочем времени летных специалистов авиакомпании.
Входные данные приложение загружает из файла, в котором хранится список летных специалистов и список перелетов, которые были выполнены самолетами авиакомпании. Для каждого перелета указывается тип и номер воздушного судна, время взлета и посадки, названия аэропортов вылета и назначения, а также список экипажа, выполнявшего перелет.
Важно:
Приложение считывает данные из файла, после чего формирует выходной файл в формате XML или JSON (на выбор). В выходном файле для каждого летного специалиста должно быть указано полетное время за каждый месяц, указанный во входном файле (то есть, если во входном файле указаны перелеты, выполненные в декабре 2020, январе 2021 и феврале 2021, для каждого специалиста должно быть указано полетное время за декабрь 2020, январь 2021 и февраль 2021).
Также, для каждого месяца у каждого специалиста должны быть особые отметки в том случае, если суммарное рабочее время превышает 80 часов или в данном месяце были недели, когда суммарное рабочее время превысило 36 часов, или в данном месяце были дни, когда суммарное рабочее время
превысило 8 часов.
Задача должна быть решена исключительно средствами Java SE, код приложения должен содержать unit-тесты, приложение должно собираться при помощи Maven или Gradle.

## Описание проекта

Консольное приложение для расчёта рабочего времени летных специалистов авиакомпании.

Приложение:

* загружает данные о летных специалистах и выполненных перелётах из файла в формате **JSON** или **XML**;
* рассчитывает полётное время каждого специалиста по месяцам;
* определяет превышение нормативов:

  * более **80 часов** за месяц;
  * более **36 часов** за неделю;
  * более **8 часов** за день;
* формирует отчёт в формате **JSON** или **XML**.

Проект реализован **исключительно средствами Java SE** без использования сторонних библиотек.
